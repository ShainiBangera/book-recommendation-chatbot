# Book Recommendation Chatbot (Rule-Based Chatbot)

This is a rule-based chatbot designed to simulate a student and college library assistant conversation.
The chatbot allows users to browse book categories, issue books, return books, and calculate fines for late returns.

## Features

-Greeting and exit interaction
-View available book categories
-Display books based on selected category
-Select and issue books (maximum 3 books)
-Return issued books
-Fine calculation for late return
-Exception handling for invalid inputs

## Conclusion

This demonstrates a simple chatbot that performs basic library operations using Python.
